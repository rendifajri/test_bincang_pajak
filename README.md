## How to install
- Open cmd, type "composer install"
- Copy .env.example to .env
- type "php artisan key:generate"
- Create database test_bincang_pajak
- type "php artisan migrate"
- type "php artisan db:seed"
- type "mkdir C:\laragon\www\rendi\test_bincang_pajak\public\vendor"
- type "ln -s C:\laragon\www\rendi\test_bincang_pajak\vendor\components C:\laragon\www\rendi\test_bincang_pajak\public\vendor\components"
- type "ln -s C:\laragon\www\rendi\test_bincang_pajak\vendor\twbs C:\laragon\www\rendi\test_bincang_pajak\public\vendor\twbs"
